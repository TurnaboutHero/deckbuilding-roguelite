# P9 신규 기획 동기화 결정

## 목적

Coin Combat 프로젝트의 최신 기획 대화 7개를 현재 구현과 대조하고, 충돌하거나 모호한 규칙을 하나의 구현 기준으로 확정한다.

## 원본 우선순위

1. 각 대화의 가장 최근 확정 문장
2. P7에서 실제 플레이테스트를 거쳐 확정한 전투 규칙
3. 기존 저장 데이터와 콘텐츠 ID 호환성
4. 구현이 필요한 모호한 부분은 가장 단순하고 예측 가능한 규칙

## 공통 유지 규칙

- 턴당 행동 횟수 제한은 두지 않는다.
- 기본 공격과 기본 방어는 반복 사용할 수 있다.
- 기본 공격은 `피해 4 / 앞면 +3`, 기본 방어는 `방어 4 / 뒷면 +3`을 유지한다.
- 스킬 슬롯은 기본 8개다.
- 속성 동전은 앞면과 뒷면 모두 고유 효과를 가진다.
- 소비형 스킬은 동전을 플립하지 않으며 동전의 면 효과도 발동하지 않는다.
- 전투당 1회와 사용 후 제거는 UI에서 `일회성`으로 통일한다.

## 화염 전사

- 시작 스킬을 `기본 공격 / 기본 방어 / 불타는 일격 / 잿불 베기`로 변경한다.
- `내면의 열정`은 시작 스킬에서 제외하되 전사 보상 풀에 유지한다.
- 저장 호환성을 위해 기존 `flame-hook` ID를 `잿불 베기`에 재사용한다.
- 잿불 베기는 코스트 1, 피해 5, 앞면 피해 +2 및 화상 2, 화염 동전 사용 시 화상 +1로 확정한다.

## 번개 결투사

- 기존 임시 번개 캐릭터 ID `sorcerer`를 유지하고 표시 정체성을 `번개 결투사`로 교체한다.
- 시작 스킬은 `기본 공격 / 기본 방어 / 아따끄 / 빠라드`다.
- 고유 패시브 `르미즈`는 상태를 저장해 다음 스킬을 강화하는 과열형 메커니즘이 아니다.
- 매 턴 첫 플립형 스킬의 첫 동전이 앞면이면 그 동전의 앞면 효과를 적용한 뒤 한 번 재플립한다.
- 재플립 결과의 면 효과도 적용한다. 재플립이 다시 앞면이면 원래 투입한 모든 동전을 다시 플립해 같은 스킬을 비용 없이 한 번 재사용한다.
- 재사용은 코스트를 추가로 내지 않고 쿨타임도 한 번만 시작한다. 재사용된 스킬은 르미즈를 다시 발동하지 않는다.
- 기본 재사용 확률은 `0.5 × 0.5 = 25%`다.
- 재플립과 재사용은 별도 이벤트로 기록해 UI 피드백과 테스트에서 관찰할 수 있어야 한다.
- 최신 대화의 르미즈 계열 5개, 감전 처형 계열 5개, 전용 패시브 8개를 추가한다.
- 최신 희귀 스킬이 요구하는 쿨타임 4를 허용하도록 타입과 검증 범위를 확장한다.
- 기존 번개 스킬 ID는 저장 호환을 위해 삭제하지 않고 레거시 보상 콘텐츠로 유지한다.

## 마도기사 병기 출력

- `병기 출력`은 전투 시작 시 0, 최대 5이며 전투 동안 유지되고 소비되지 않는다.
- 현재와 이후에 소환되는 모든 장비의 효과에 병기 출력만큼을 더한다.
- 기존 개별 소환 강화 수치와 병기 출력은 합산한다.
- 최신 병기 출력 스킬 5개를 추가하고 기존 임시 병기 스킬은 저장 호환을 위해 유지한다.
- 확산 각인의 광역 공격 횟수와 복제된 장비의 임시 강화는 소환 상태에 명시적으로 저장한다.
- 소환 한도가 찬 상태의 복제는 가장 오래된 소환을 밀어내지 않고 선택한 소환의 지속 시간을 늘린다.
- `청람 병장개문`은 4번째 이상의 영구 소환 상태를 만들지 않는다. 현재 소환 수에 따라 계산한 임시 마나 검 공격을 즉시 실행하고 소멸시킨다.

## UI와 저장 호환

- 전투 UI에 병기 출력 0~5를 표시한다.
- 르미즈의 이번 턴 사용 가능 여부, 재플립, 무료 재사용을 전투 피드백에서 구분한다.
- 소환 카드에는 지속 시간, 개별 강화, 남은 광역 공격 횟수를 표시한다.
- 캐릭터와 스킬 ID 유지로 현재 런 저장 스키마 버전은 올리지 않는다.
- 새 전투 상태는 런 저장 데이터가 아니라 전투 시작 시 초기화되는 상태다.

## 테스트 기준

- 전사 새 런의 시작 스킬 4개와 내면의 열정 보상 등장 가능성을 검증한다.
- 르미즈의 미발동, 재플립, 25% 경로의 무료 재사용, 쿨타임 1회, 코인 1회 폐기를 고정 시드로 검증한다.
- 감전 배가, 유지, 처형, 실패 시 피해 및 제거 규칙을 검증한다.
- 병기 출력의 0~5 상한, 기존/신규 소환 적용, 명령 적용, 복제 한도 대체 규칙을 검증한다.
- 카드 설명과 캐릭터 선택 화면에 새 규칙이 `특수` 같은 대체 문구 없이 표시되는지 검증한다.

## 강화 확정 및 미확정

- 연성 참격: 병기 출력 획득 조건을 `뒷면 1개 이상`에서 `뒷면 1개마다`로 강화한다.
- 확산 각인: 앞면과 뒷면이 모두 나오면 임시 마나 동전 1개를 손패에 추가한다.
- 마력로 과급: 소비 마나 동전 2개를 1개로 줄인다.
- 마도식 복제: 소환 한도 도달 시 선택 장비 지속 증가량을 2에서 3으로 높인다.
- 청람 병장개문: 임시 마나 검의 기본 피해를 3에서 4로 높인다.
- 레두블망: 추가 르미즈 기회를 1회에서 2회로 높인다.
- 아타크 콩포제: 무료 재사용 후 생성하는 임시 번개 동전을 1개에서 2개로 높인다.
- 전하 각인: 앞면 감전을 1에서 2로 높인다.
- 축전 방패: 기본 방어를 7에서 8, 감전 참조 상한을 5에서 8로 높인다.
- 과전도: 일회성을 제거하고 쿨타임 4로 변경한다.
- 과부하 연격: 기본 피해를 4에서 6으로 높인다.
- 뇌정 처형: 소비 번개 동전을 3개에서 2개로 줄인다.
- 아따끄, 빠라드, 팡트, 파라드-리포스트, 플레슈의 강화 효과는 최신 대화에서 확정되지 않았다. 임의 효과를 만들지 않고 미확정으로 남긴다.

## 미확정 사항

- 최신 추출에서 `아따끄`, `빠라드`, `팡트`, `파라드-리포스트`, `플레슈`의 강화 효과는 명시되지 않았다.
- 위 5개 스킬은 임의 강화 효과를 만들지 않고 강화 정의를 비워 둔다. 후속 기획에서 수치와 방식을 확정한 뒤 추가한다.
