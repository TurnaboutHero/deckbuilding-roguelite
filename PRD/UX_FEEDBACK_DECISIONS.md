## 바꾸지 않은 것

- **모든 전투 수치·확률·적 패턴·스킬 값·코인 구성** — 사용자도 "수치 밸런스보다 정보 문제"로 진단했고, 수치는 사람 플레이 로그(N≥5) 판정 전 변경 금지.
- 픽셀 UI 미학, 컴팩트 HUD(≤78px), 카드 폭(≤126px), 조작 체계(클릭+드래그 이중 양식), 시드 결정론(seed 42 골든 불변).
- 새 이미지 에셋 없음 — CSS만으로 충분해 sprite-gen/공냥 프롬프트 킷 미사용.
- Phase 3 보류·사람 게이트 지위 (`PRD/PHASE1_HOLDS.md`) — 이 라운드는 어느 게이트도 대신 닫지 않는다.
