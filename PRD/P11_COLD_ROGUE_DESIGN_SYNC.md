# P11 냉기 도적 기획 동기화

> 동기화일: 2026-07-13 · 콘텐츠 버전 `1.5.0-p11` · 상태: engineering-complete / experience-unverified

## 확인된 원본

- Coin Combat 프로젝트의 최신 `냉기 도적 스킬 기획` 대화를 기준으로 한다.
- 기존 저장 호환을 위해 캐릭터 ID `frost-knight`는 유지하고 표시명과 콘텐츠를 `냉기 도적`으로 교체한다.
- 냉기 동전은 앞면 `동상 1`, 뒷면 `방어 1`의 기존 확정 규칙을 유지한다.

## 확정 구현

- 고유 특성 `이중 주머니`: 턴 종료 시 손의 동전 1개를 보존한다. 보존 표식은 사용 또는 소비 전까지 유지된다.
- 추가 보존은 턴당 최대 2개, 전투 중 동시 보존은 최대 3개다.
- 지정 뽑기는 기본/냉기 중 선택한 실제 동전을 뽑을 더미에서 찾고, 없으면 아무 일도 일어나지 않는다.
- 냉기 소비 비용은 실제 냉기 동전만 지불할 수 있다. 냉기로 취급되는 기본 동전은 플립/속성 상호작용에만 유효하다.
- 냉기 동전 소비 시 플립과 고유 면 효과는 발생하지 않는다.
- 시작 스킬은 공용 `베기`, `방어`와 `얼음 발톱`, `얼음 밑장빼기`다.
- 시작 전용 2종, 동결 건조 암살자 5종, 빙점 밀수꾼 5종으로 냉기 전용 스킬을 정확히 12종만 운영하고, 전용 패시브 8종과 강화안을 반영한다.
- 구기획 보상 스킬 `동장군(winters-grasp)`은 현 보상 풀에서 제거한다. P10 저장에만 남은 ID는 `서리털 망토(frost-fur-cloak)`로 마이그레이션한다.

## 구현 판단

- 체력은 새 수치가 없으므로 기존 냉기 캐릭터의 최대 체력 70을 상속한다.
- 동상은 현재 엔진 규칙(공격 피해 25% 감소, 턴 경과 시 1 감소)을 상속한다.
- 지정 뽑기 선택 UI가 열리지 않는 자동 진행·시뮬레이션에서는 뽑을 더미에 실제 존재하는 후보를 선언 순서대로 선택한다.
- 전투 시작 `감별사의 엄지`는 별도 선택 화면이 없는 현재 UI에서 냉기 우선, 없으면 기본 동전을 뽑는다.
- `비상용 얼음주머니`는 기본 상태에서 전투당 1회 잠금이 우선하고 쿨타임 3을 함께 명시한다. 강화로 일회성이 제거되면 같은 쿨타임 3으로 복귀한다.
- `숨은 안주머니`, `장물 바꿔치기`처럼 동전 자체를 보존하는 효과는 그 동전만 보존하며 추가 보존 용량을 늘리지 않는다. `이중 주머니`의 기본 선택 1개는 별도로 유지된다.
- `서리 복리`의 피해 +3은 기존 공격의 첫 피해 축에 합산하며 별도 추가 타격을 만들지 않는다.
- `차가운 손버릇`이 보존 냉기 동전 사용으로 동상을 부여한 뒤 `서리 복리`를 판정한다. 보존 기본 동전을 냉기로 취급하는 스킬도 이 플립 상호작용에 포함한다.
- 냉기 소비 UI는 `일부 소비`를 1~최대 수량으로, `전부 소비`를 최소 조건 충족 후 손의 실제 냉기 전부로 표시·선택한다. 보존 여부가 효과를 바꾸는 1개 소비도 대상 동전을 직접 고르며, 속성과 무관한 화염 아이콘은 사용하지 않는다.
- 전투 UI에서 `턴 종료`를 처음 누르면 보존 선택 모드가 열린다. 손패와 장전된 정리 후보를 토글하고 버튼 재클릭 또는 Enter로 확정하며 Escape로 취소한다.
- 자동 진행에서는 이미 보존된 동전을 유지한 뒤 손패 순서대로 합법 범위만 자동 선택한다.

## 미확정 및 보류

- 혈액 속성 캐릭터의 전용 기획 대화는 Coin Combat 프로젝트 목록과 확인 가능한 대화 기록에서 찾지 못했다. 이름, 고유 메커니즘, 시작 스킬, 전용 패시브를 임의로 만들지 않고 후속 기획 확인까지 보류한다.
- 냉기 도적 전용 신규 캐릭터/카드 아트는 확정 원본이 없으므로 기존 냉기 캐릭터 아트를 임시 재사용한다.
- 지정 뽑기의 별도 선택 모달은 후속 UX 개선 대상으로 남긴다. 턴 종료 보존은 핵심 전략이므로 P11에서 직접 선택 UI를 제공한다.
