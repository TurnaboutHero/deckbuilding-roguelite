# P10 화염 전사·마도기사 최신 기획 동기화

## 목적

2026-07-13 기준 ChatGPT 프로젝트의 `화염 전사 스킬 기획`, `마도기사 스킬 기획` 최신 통합 응답을 현재 P9 구현과 다시 대조한다. 확정된 규칙과 수치는 구현하고, 최신 응답에서 미정으로 남긴 항목은 임의로 채우지 않는다.

## 적용 우선순위

1. 두 대화의 가장 최근 통합 기획서
2. P9에서 이미 검증한 전투 엔진 계약과 저장 호환성
3. 기존 콘텐츠 ID를 유지할 수 있는 경우 ID 유지
4. 기획서가 명시적으로 미정이라 한 항목은 구현에서도 강화 정의를 두지 않음

## 공통 확정

- 공통 공격은 코스트 1, 반복, 피해 4, 앞면 피해 +3이다.
- 공통 방어는 코스트 1, 반복, 방어 4, 뒷면 방어 +3이다.
- 소비형 스킬은 지정 속성 동전을 플립하지 않고 소비한다.
- 일회성 스킬은 해당 전투에서 한 번만 사용할 수 있다.
- 화상과 방어처럼 참조되는 수치는 참조 효과로 인해 감소하지 않는다.

## 화염 전사

### 시작 세트

- `공격 / 방어 / 불타는 일격 / 불씨 베기`의 4개를 유지한다.
- `불타는 일격`은 코스트 2, 피해 8, 앞면마다 피해 +3, 버림 더미에 임시 화염 동전 1개 생성이다.
- `불씨 베기`는 코스트 1, 피해 5, 앞면 피해 +2와 화상 2, 화염 동전을 사용했다면 화상 +1이다.
- 저장 호환을 위해 P9의 `burning-fist`, `flame-hook` ID를 유지한다.

### 화상 빌드

- `잿불 갑주`: 방어 4, 앞면이면 손패에 임시 화염 동전 1개, 뒷면이면 방어 +3, 화염 동전을 사용했다면 방어 +1.
- `화염 폭주`: 일회성 지원기. 코스트 2, 손패에 임시 화염 동전 2개, 모든 적에게 화상 3.
- `홍련 회전참`: 코스트 2 광역 공격. 피해 3, 앞면마다 광역 피해 +2, 뒷면마다 모든 적 화상 1, 앞면 화염 동전마다 모든 적 화상 +2.
- `업화의 처형`: 화염 동전 3개 소비, 일회성. 피해 `6 + 대상 화상 × 3`, 화상은 유지.

### 과열 빌드

- 과열은 중첩되지 않고 턴을 넘어 유지된다.
- 과열 보너스가 실제로 적용된 스킬의 처리가 끝난 뒤에만 해제된다.
- `내면의 열정`은 시작 세트가 아닌 일반 획득 스킬이다. 화염 동전 1개를 플립 비용으로 요구하고, 과열 진입 후 앞면이면 피해 5, 뒷면이면 방어 3을 얻는다.
- `화격권`은 코스트 2, 피해 10, 과열이면 총 피해 14, 앞면마다 피해 +1, 화염 동전 앞면은 합계 +2, 뽑기 더미에 임시 화염 동전 1개를 생성한다.

### 화염 전사 패시브

- 무속성: 강철 피부, 방패 숙련, 빈틈없는 대비, 불굴의 투지, 전투 호흡.
- 화염: 발화 본능, 잔불 칼날, 뜨거운 방벽.
- 각 효과는 최신 통합안의 1줄 설명과 발동 제한을 그대로 사용한다.

## 마도기사

### 시작 세트와 장비

- 시작 세트는 `공격 / 방어 / 마력 충전 / 명령`이다.
- `마력 충전`은 코스트 1, 반복. 선택한 마나 검 또는 마나 방패를 소환 2로 만들고 뒷면마다 소환 +1, 임시 마나 동전 1개를 생성한다.
- `명령`은 코스트 2, 반복. 선택 장비를 즉시 행동시키고 소환 -1, 뒷면마다 이번 행동 수치 +1, 버림 더미에 임시 마나 동전 1개를 생성한다.
- `마도 공방`은 빈 소환 슬롯이 있을 때 매 턴 마나 검을 소환 1로 만든다.
- 장비 상한은 3이며 마나 검은 피해 3, 마나 방패는 방어 2의 턴 종료 행동을 한다.

### 마력 갑주 빌드

- `마력 반격`: 피해 8, 앞면마다 피해 +3, 뒷면마다 방어 +2.
- `갑주 축압`: 방어 7, 앞면마다 이번 턴 다음 공격 피해 +2, 뒷면마다 방어 +3.
- `마력 증폭막`: 마나 동전 2개 소비, 현재 방어만큼 방어 획득, 최대 10.
- `갑주 강타`: 마나 동전 2개 소비, 피해 `6 + min(현재 방어, 10)`.
- `마도 갑주 해방`: 마나 동전 3개 소비, 일회성. 방어 10을 얻고 장비 행동 뒤 모든 적에게 현재 방어만큼 피해, 최대 18.

### 마나 병기 빌드

- 병기 출력은 전투 시작 시 0, 최대 5, 전투 동안 유지되고 소비되지 않는다.
- 마나 검 피해와 마나 방패 방어에 병기 출력을 더한다.
- `연성 참격`, `확산 각인`, `마력로 과급`, `마도식 복제`, `청람 병장개문`은 P9 ID와 효과를 유지한다.
- `청람 병장개문` 강화는 검 한 개의 피해를 높이는 것이 아니라 기본 임시 검 수를 `3 + 장비 수`에서 `4 + 장비 수`로 높인다.

### 마도기사 패시브

- 무속성: 전개 예습, 역상 방호식, 교차 연산, 잔류식 재구축, 명령 보존식.
- 마나: 마나 피막, 청색 순환로, 병장 공명로.
- 패시브는 최신 통합안의 턴당/전투당 제한과 최대치까지 포함해 구현한다.

## 충돌 해소

- 화염 전사 통합안의 "무속성과 화염 동전만 사용"은 화염 전용 소비 비용과 시너지 범위를 뜻하는 캐릭터 설계 제약으로 적용한다. 게임의 공통 보상에서 다른 속성 동전을 획득할 수 있는 기존 규칙은 유지한다.
- `내면의 열정`은 플립형으로 명시됐으므로 P9의 소비형 대체안과 드로우 보상은 제거한다. 화염 동전을 실제 플립 비용으로 요구하는 계약을 사용한다.
- `마력 충전`의 임시 마나 동전 생성 위치는 최신안에 위치가 없으므로 기존 P9 동작인 손패 생성을 유지한다.
- 기존 런 저장을 깨지 않기 위해 기존 스킬·패시브 ID는 삭제하지 않는다. 최신안으로 의미가 직접 교체되는 시작 스킬은 같은 ID를 갱신하고, 별도 획득 스킬은 신규 ID를 추가한다.
- 화염 전사는 검사가 아닌 격투가 판타지를 최종 표현으로 사용한다. 따라서 기획 작업명은 저장 ID와 추적용으로 유지하되, 플레이어 표시명은 `불타는 일격→불꽃 스트레이트`, `불씨 베기→불씨권`, `잿불 갑주→잿불 위빙`, `홍련 회전참→회전 연화각`, `업화의 처형→폭렬권`으로 통일한다.

## 미확정 사항

- `불씨 베기` 강화는 최신 통합안에서 미정이다.
- `내면의 열정` 강화는 기존 쿨타임 감소안이 폐기됐으며 대체안은 미정이다.
- 과열 전용 패시브는 최신 통합안에 확정안이 없다.
- 위 항목은 강화나 패시브를 임의 생성하지 않는다.

## 검증 기준

- 새 화염 전사 런의 시작 4스킬, 화염 조건부 플립, 과열 유지·소비, 화상 비소비를 회귀 테스트한다.
- 새 마도기사 런의 시작 코스트, 선택 소환, 명령, 갑주 방어 참조, 병기 출력 상한과 임시 검 수 강화를 회귀 테스트한다.
- 전용 패시브의 발동 제한과 누적 상태를 고정 시드로 검증한다.
- 콘텐츠 검증, 타입 검사, 린트, 전체 테스트, 빌드, 시뮬레이션과 브라우저 플레이테스트를 모두 통과해야 한다.
