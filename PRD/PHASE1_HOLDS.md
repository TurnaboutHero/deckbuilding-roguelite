# Phase 1 보류 원장

> 기록일: 2026-07-10  
> 결정: Phase 1의 코드 구현·자동 검증·배포는 완료 상태를 유지한다. 아래 항목은 **통과로 간주하지 않고 보류**하되, Phase 2(M5~M6) 자동 개발의 선행 차단 조건으로 사용하지 않는다.

| 항목 | 현재 상태 | 닫힘 조건 |

## 유지하는 MVP 가정

- 사운드는 MVP 범위에서 제외한다.
- 한국어 단일 언어로 제공한다.
- 최신 데스크톱 브라우저를 우선 지원한다.

## 운영 규칙

- 위 항목은 닫힘 증거가 생기기 전까지 `완료` 또는 `통과`로 바꾸지 않는다.
- Phase 2 구현 중 관련 코드를 건드리더라도 이 원장의 범위를 몰래 확장하거나 판정을 대신하지 않는다.
- M6 사람 지표 게이트와 Phase 3 진입 판정은 이 예외에 포함되지 않는다. M6 자동 지표와 사람 표본을 확인한 뒤 별도로 결정한다.
