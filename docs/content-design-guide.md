# 콘텐츠 기획 가이드 v1.0 — 캐릭터 · 스킬 · 몬스터 양식

> 새 캐릭터/스킬/몬스터를 기획할 때 그대로 복사해 쓰는 양식과 수치 기준.
> 규칙의 원본은 [PRD.md](./PRD.md)(v0.4)와 [implementation-plan.md](./implementation-plan.md)(v1.2). 이 가이드의 수치 기준은 전부 **시뮬레이터(M6) 튜닝 대상 초안**이다.

---

## 0. 공통 행동 규칙 (모든 기획의 전제)

- 매 턴 코인 5개 드로우
- **한 턴에 스킬 최대 3회 사용** (장전형 + 소비형 합산)
- 각 스킬 카드는 턴당 1회
- 미사용 코인은 턴 종료 시 폐기 (이월 없음)
- **일반 스킬 최대 코스트 4. 5코스트는 희귀·일회성·궁극 전용. 6코스트 이상 금지**
- 소비형 조건은 속성 코인 **1~3개**만 (4개 이상 금지)
- 큰 기술은 높은 코스트가 아니라 **조건/충전/소비형**으로 표현한다
  (예: "화염 코인 3개 소비", "이번 전투에서 화상 12 이상 부여했으면 사용 가능", "전투 중 충전 3회 누적")

턴당 3회 제한의 근거 — 5코인을 다 쓰려면 스킬당 평균 코스트:

| 턴당 제한 | 평균 코스트 | 판단 |
|---|---|---|
| 2회 | 2.5 | 초반 스킬이 답답함 |
| **3회** | **1.67** | **적정 — 코스트 관리 감각 유지** |
| 4회 | 1.25 | 1코스트 난사 느낌 |
| 5회 | 1.0 | 사실상 무제한 |

시작 키트(1/1/2/1코스트)로는 3스킬 = 4코인 사용이 자연값 → **코인 1개가 남는 구조가 2~3코스트 스킬 채용 유인**이 된다. 코인 낭비율은 시뮬 관측 지표.

## 1. 코스트 → 기대값 기준표 (스킬 위력의 자)

| 코스트 | 피해 기대값 | 방어 기대값 | 역할 | 추천 등급 |
|---|---|---|---|---|
| 1 | 7~9 | 5~7 | 가벼운 기본 행동 | 일반 |
| 2 | 10~13 | 9~12 | 핵심 주력기 | 일반/고급 |
| 3 | 15~19 | 13~17 | 강한 행동 | 고급/희귀 |
| 4 | 20~26 | — | 턴의 핵심 행동 | 희귀 |
| 5 | 28+ 또는 판을 바꾸는 효과 | — | 올인기 | 희귀 + **일회성/궁극 전용** |

**엔진 패널티 원칙**: 코인 생성·변환·손패 조작 같은 엔진 효과가 붙으면 즉시 수치를 기준표 하단 이하로 낮춘다. (불타는 일격: 2코스트 기대 11 — 순수 공격기 상한 13보다 낮은 대신 임시 화염 코인 생성)

기준점 (확정 스킬): 베기 EV 8 (1코스트) · 방어 EV 6.5 · 불타는 일격 EV 11+α (2코스트) · 점화 검술 13 확정 (화염 1 소비)

## 3. 캐릭터 기획 양식

원칙: 고유 특성은 **패시브 1개**, 전투 시작/턴 시작/속성 코인 사용 같은 자동 조건으로 발동. 수치 버프(공격 +2 같은)가 아니라 **코인 규칙을 살짝 바꾸는 효과**여야 한다.

```
[캐릭터명]
대표 속성:
시작 HP:                     ← 기준: 전사 70
시작 코인:                   ← 기준: 기본 8 + 대표 속성 2
시작 스킬:                   ← 4종 (슬롯 6 중 4 채움)
고유 특성명:
고유 특성 효과:
발동 조건:                   ← 전투 시작 시 / 턴 시작 시 / ~할 때마다
발동 횟수 제한:
전투 중 생성/변환 여부:      ← 생성 코인은 반드시 임시
주요 빌드 방향:
약점:
```

### 확정: 전사

```
[전사]
대표 속성: 화염
시작 HP: 70
시작 코인: 기본 8, 화염 2
시작 스킬: 베기 / 방어 / 불타는 일격 / 점화
고유 특성명: 불씨 주머니
고유 특성 효과: 전투 시작 시 임시 화염 코인 1개를 뽑을 더미에 추가한다 (전투 종료 시 제거)
발동 조건: 전투 시작 시 자동
발동 횟수 제한: 전투당 1회
주요 빌드 방향: 화염 코인 생성 → 화상 누적 → 소비형 확정 고점
약점: 방어 수단이 방어 스킬 1종뿐 — 장기전에 취약
```

효과 검증: 첫 손패 화염 ≥1 확률 77.8% → **87.9%**, ≥2 확률 22.2% → **42.4%** (§2 표). 정체성은 강화되지만 확정 공급은 아님 — 적정.

## 4. 스킬 기획 양식

### 전사 화염 빌드의 역할 분류 — 모든 화염 스킬은 아래 중 하나 이상의 역할을 가진다

| 역할 | 설명 | 확정 예 |
|---|---|---|
| 피해형 | 즉시 피해 | 베기 |
| 화상 부여형 | 화상 스택 축적 | 점화 |
| 생성형 | 임시 화염 코인 생성 | 불타는 일격, 화염 주입 |
| 소비형 | 화염 코인을 플립 없이 소비해 확정 효과 | 점화 검술, 화염검 |
| 변환형 | 기본 코인을 이번 턴 화염 취급 | 용광로, 화염 폭주 |
| 증폭형 | 쌓인 화상을 활용한 추가 가치 | (미확정 — 확장 슬롯) |

```
[스킬명]
캐릭터:                 대표 속성:
스킬 등급: 일반/고급/희귀
스킬 타입: 장전형 / 소비형 / 지원형
코스트: (장전형: 코인 n / 소비형: 속성 코인 n개 소비)
대상: 적 1명 / 모든 적 / 자신 / 없음
사용 제한: 턴당 1회 (기본) / 전투당 1회 (일회성)
기본 효과:              ← 반드시 존재 ("스킬은 실패하지 않는다")
앞면 추가 효과:          ← 표기 규칙: "앞면: X"=1회, "앞면 1개마다 X"=코인당
뒷면 추가 효과:
코인 조작: 생성(→버림/뽑을 더미/손패) / 변환 / 소비
상태이상: 화상 / 동상 / 감전 / 위축
빌드 역할: (위 분류표에서)
기대값: (§1 기준표와 대조 — 엔진 효과 있으면 하단 이하)
리스크:
시너지:
카드 문구: (효과 텍스트는 데이터에서 자동 생성 — 문구는 검수용)
```

### 예시 — 확정 4종 요약

| 스킬 | 등급/타입 | 코스트 | 효과 요약 | 역할 | 기대값 |
|---|---|---|---|---|---|
| 베기 | 일반/장전형 | 1 | 피해 6, 앞면 +4 | 피해형 | 8 |
| 불타는 일격 | 일반/장전형 | 2 | 피해 8, 임시 화염→버림, 앞면 1개마다 +3 | 생성형 | 11+α |
| 점화 검술 | 고급/소비형 | 화염 1 소비 | 피해 10, 화상 +2 (확정) | 소비형 | 13 |
| 화염 폭주 | 희귀/장전형·**일회성** | 1 (**전투당 1회**) | 손패 기본 코인 전부 화염 취급, 앞면: 임시 화염 손패 +1, 뒷면: 자신 피해 2 | 변환형 | 콤보 턴 생성 |

> 화염검(소비형, "이번 턴 피해를 줄 때마다 화상 +1")의 소비 조건은 **v0.4에서 1개(고급)로 확정** — 턴당 3회 제한으로 발동 후 공격 기회가 최대 2회. PRD §8.3 반영 완료. 3개 소비 계단은 불의 심장(희귀, 공격 스킬마다 화상 +2)이 담당.

## 5. 몬스터 기획 양식

### 원칙

1. 기본은 **순서형 행동 루프** — 패턴 학습 가능, 의도 표시가 의미를 가짐, 밸런스 조절 용이. 완전 랜덤은 특정 몬스터의 개성으로만.
2. 매 턴 의도 공개 (행동 타입 + 수치).
3. **다중 적 전투에서 전원이 매 턴 공격 금지** — 공격/방어/디버프로 행동을 분산해 "읽히는 전투"를 만든다.
4. 상태이상은 확정 어휘만 사용: **화상 / 동상(가하는 피해 25%↓) / 감전(받는 피해 50%↑) / 위축(드로우 −1)**. "취약·약화" 같은 외부 어휘는 이 넷으로 매핑한다.

### 행동 타입

| 타입 | 설명 | 의도 표시 예 |
|---|---|---|
| 공격 | 플레이어에게 피해 | ⚔ 8 |
| 연속 공격 | 다단 피해 | ⚔ 4×2 |
| 방어 | 자신에게 방어 | 🛡 6 |
| 버프 | 자신 강화 (충전 포함) | ↑ 강화 |
| 디버프 | 플레이어에게 상태이상 | ☠ 위축 |
| 충전 | 다음 턴 강공격 예고 | ⚡ 충전 |
| 소환 | 적 추가 (Phase 3) | ✚ 소환 |
| 특수 | 규칙 개입 | ★ |

```
[몬스터명]
등급: 일반 / 엘리트 / 보스
역할: 공격형 / 방어형 / 디버프형 / 지원형 / 폭발형
체력:                        ← §5.1 밸런스 기준표 참조
등장 구간:                   ← 전투 1~5 중
동시 등장 가능 수:
패턴 방식: 순서형 루프 (기본) / 조건부 혼합 / 랜덤(개성용)
행동 루프:
  1턴: [행동명] — [타입] [수치] [부가 효과]
  2턴: …
  반복 방식:
조건부 행동: (체력 50% 이하 / 특정 턴 도달 시 등 — Phase 3)
의도 표시: (턴별 아이콘+수치)
긴장감 포인트:
플레이어 대응법:
```

### 5.1 몬스터 밸런스 기준 (플레이어 HP 70 · 턴 출력 ~20 · 방어 상한 8/턴 스케일)

> ⚠️ 외부 초안의 "플레이어 HP 25, 몬스터 HP 6~14" 스케일은 확정 스킬 기대값(턴당 ~20 피해)과 모순되어 **기각**. 아래는 확정 스케일로 환산한 기준이다.

| 전투 구성 | 적 합계 HP | 턴당 적 피해 합 | 비고 |
|---|---|---|---|
| 단일 일반 (MVP) | 55~75 | **공격형 8~11 / 방어·디버프형 4~7** (피해 대신 방어·상태이상으로 압박) | 약탈자 75 (10/턴) / 수문장 70 (5/턴+방8) / 주술사 60 (평균 4.5/턴+위축) 확정 |
| 일반 2마리 | 65~85 | 9~13 | 행동 분산 필수 — 조합으로 대역 충족 (예: 고블린+구울 = HP 70, 피해 ~10.7/턴) |
| 일반 3마리 | 75~95 | 10~14 | 전원 공격 턴 금지 (예: 구울+고블린+슬라임 = HP 86) |
| 엘리트 | 90~110 | 12~16 | |
| 보스 강공 턴 | 150+ | 18~24 | 충전으로 반드시 예고 |

하한 가드: **일반 적 개체 HP ≥ 12** (한 스킬로 증발 방지), **단일 적 HP ≥ 55** (폭주 콤보 원턴킬 가드).

### 5.2 다중 적 전투용 몬스터 예시 5종 (Phase 3 콘텐츠 초안 — 확정 스케일 환산)

| 몬스터 | HP | 역할 | 행동 루프 (반복) | 대응법 |
|---|---|---|---|---|
| 고블린 | 32 | 기본 공격형 | 찌르기 ⚔7 → 몸 숨기기 🛡5 → 난도질 ⚔10 | 공격 턴에 방어, 방어 턴에 화상 축적 |
| 도적 | 26 | 방해형 | 기습 ⚔6 → 약점 찌르기 ⚔6+감전(1턴) → 회피 🛡7 | 감전 턴 전에 처치 또는 방어 |
| 구울 | 38 | 장기전 압박 | 썩은 손길 ☠동상(1턴) → 물어뜯기 ⚔8 → 포식 ⚔7+자가 회복 5 | 화상으로 회복 상쇄 |
| 마도사 | 22 | 충전형 | 주문 충전 ⚡(다음 공격 +5) → 화염탄 ⚔11+화상1 → 보호막 🛡8 | 충전 턴에 집중 처치 |
| 슬라임 | 16 | 연계 방해 | 들러붙기 🛡4 → 산성 점액 ⚔5+위축 → 튀어오르기 ⚔8 | 초반 제거로 디버프 차단 |

원 초안의 상태이상 매핑: 취약 → **감전**, 약화 → **동상**, 방어 감소 → 삭제(어휘 미확정, 감전으로 대체). HP는 §5.1 다중 적 합계 대역(2마리 65~85, 3마리 75~95)에 조합이 들어오도록 환산했다 — 개체 하한 가드(HP ≥ 12)도 전원 충족.

## 6. 성장 구조 — 상점 · 재화 · 이벤트 (Phase 2~3 스펙)

성장의 두 축: **스킬 성장**(구매/보상) × **코인 성장**(추가·제거·변환 = 주머니 확률 조작).

### 상점 (재화: 골드 — 적 처치/이벤트로 획득)

| 행동 | 내용 | 기준 가격 (초안) |
|---|---|---|
| 원하는 동전 제거 | 주머니에서 1개 영구 제거 | 75 → 구매마다 +25 (75/100/125/150…) |
| 스킬 구매 | 무작위 5종 진열 (일반 3 + 고급 1 + 무작위 1, 후반일수록 희귀 확률↑) | 일반 50 / 고급 80 / 희귀 120 |
| 동전 구매 | 무작위 3종 진열 (대표 속성 가중 + 보유 속성 보정 — "빌드를 따라오는 랜덤") | 기본 25 / 대표 속성 50 / 타 속성 70 |

골드 수급 초안: 일반 전투 30~45, 엘리트 60~80. (가격과 함께 시뮬 튜닝 대상)

### 이벤트 원칙 — **보상에는 반드시 위험요소가 있다**

스킬/동전을 공짜로 주는 이벤트 금지 (전투 보상·상점의 가치 보호). 위험요소는 넷 중 하나:

| 위험요소 | 예 |
|---|---|
| 전투 | 엘리트 전투 후 희귀 스킬 획득 |
| 체력 감소 | HP 5 잃고 화염 코인 1개 획득 |
| 재화 소모 | 100골드로 기본 코인 1개를 화염 코인으로 교체 (영구 변환은 이벤트 전용 — PRD §10) |
| 자원 소모 | 기본 코인 1개 제거 후 속성 코인 획득 / 보유 스킬 1개 포기 후 희귀 획득 |

특수 코인(저주 코인 등)은 비목표(PRD §4) — 저주류 페널티는 코인이 아닌 위 위험요소로 표현한다.
